# Project Content Folder

Is folder me har project ka markdown file rakho.

## File naming
- ai-resume-analyzer.md
- ai-code-reviewer.md

## Kaise fill karna hai
1. Har section ka content simple text ya bullets me likho.
2. Agar screenshot hai to image path do (example: /projects/ai-resume-analyzer/home.png).
3. GitHub aur Live URL exact paste karo.
4. Agar koi section missing ho to `N/A` likh do.

Jab files fill ho jayein, bata dena. Main isi content se project detail pages bana dunga.
